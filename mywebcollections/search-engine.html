2018 年各个国家的主流搜索引擎统计 - 茄子的笔记本

如果你认为除了谷歌以外，全世界只有三到四个二线地位的搜索引擎的话，那你就大错特错了。

即使把行业专用的搜索引擎排除在外，各种专业目录排除在外，仍然可以随随便便数出几百个搜索引擎。

13Tabs 印度本土搜索引擎，用户隐私至上原则，可以搜索图片，新闻，旅游目的地，旅馆信息。

Active Search Results 成立于 2009 年， 是一个独立搜索引擎，它们家采用的搜索算法还有自己的专利，叫 ASR 排名算法 ，根据用户访问数据对网站进行 asr 打分和排名.

Ant 自带书签功能，用户可以收藏它们喜欢的网站. 还提供下载和转换视频格式的服务.

Baidu 不用说了，Chinese 都知道

Bing 的前身是微软公司旗下的 msn 搜索引擎，2009 年成立，目前还是捆绑在 ie 和 edge 浏览器上自带的搜索引擎。

Cốc Cốc 越南本土 NO1 搜索引擎，在越南市场上越来越有力压谷歌的趋势了，它在对越南语的本地化处理方法比谷歌强不是一点点。

Daum 韩国门户网站兼搜索引擎

Deusu 是一家德国搜索引擎，完全由一个人开发的, 创始人叫 Michael Schöbel. Deusu’非常尊重用户隐私. 无 cookie, 无广告. Deusu 还是一个自由开源项目，你在 GitHub 上还可以找到代码，还有免费的 api 接口可以用. Deusu 搜索引擎的德语全名其实是 Deutsche Suchmaschine, 这里为了简写方便称作 Deusu, 字面意思就是 “德语的所搜引擎”。

DuckDuckGo 成立于 2008 年，强项是抓取维基百科等专业网站，另外能够结合其它搜索引擎返回适当结果. DuckDuckGo 尊重用户隐私.

EntireWeb 成立于 2000 年，由一家瑞典公司叫 Entireweb Sweden AB, 这家公司的业务就是搜索引擎，专门为第三方公司提供搜索技术解决方案。

Exalead 是一家法国公司，专注于搜索相关的各种应用。

Envolk 美国搜索引擎，尊重隐私，不用 cookie, 不存数据。

Findx 欧洲搜索引擎，尊重隐私， 源码在 GitHub 上开放.

Geliyoo 一家土耳其公司 Geliyoo Bilisim2010 年成立的搜索引擎。

Gibiru 2009 年成立，特点是不过滤搜索结果，不论隐私，受监管的还是匿名的信息，都不过滤，自己体会。

Gigablast 创立于 2000 年， by Matt Wells ，它是许多搜索引擎的搜索内容提供商。

Haosou 不解释，国产货

Istella 意大利语搜索引擎，提供新闻，图片，等信息。

Izsearch 一家通用搜索引擎，自带 50 个最热的话题分类，热点趋势，你可以直接查询分类结果。

Kaz 哈萨克斯坦搜索引擎，专注于俄语和哈萨克语的搜索。

Linkapedia 通用搜索引擎，主要从 bing 返回搜索结果，但是自家专注于从维基百科及一些专业大型信息网站上发掘信息，有自己的排名算法叫 Denis 算法.

Lookseek 匿名搜索引擎，尊重隐私

Mail.ru 毛子最流行的搜索引擎之一，肯定不如 yandex, 可以定制个性化界面. 擅长搜索文字、图片、视频和 app.

Mojeek 英国本土搜索引擎

Naver 韩国的 no1 搜索引擎，99 年成立，既是门户网又是搜索引擎。

Orange 大名鼎鼎的法国电信商，但是也提供搜索引擎服务

Parsijoo 2010 年成立的伊朗搜索引擎，专注于波斯语网站的搜索. Parsijoo 目前在伊朗排行老二。

Qwant 法国搜索引擎，有自己的搜索算法，但也结合 bing 搜索返回组合后的结果。

Seznam 捷克搜索引擎，96 年成立，在捷克共和国人气很旺。

Sogou 不说，chinese 都知道

Solofield 日本搜索引擎

Sputnik 俄罗斯搜索引擎 2014 年成立， Rostelecom 电信公司拥有. 能够搜索 图片，地图，新闻，和医疗信息

Superarama 2011 年成立的土耳其语搜索引擎. Superarama 的特点是基于 p2p 架构的搜索引擎，服务器压力小，这家搜索引擎特别注重保护年轻人，不会返回他们想要的信息。

Swobbl 德国搜索引擎，提供隐私保护。

Teoma 2000 年成立

WotBox 2013 年成立，隐私友好

Yahoo ! 雅虎搜索

Yandex 俄罗斯搜索引擎老大，同时在欧洲也是最大的搜索引擎之一。

Yooz 伊朗搜索引擎，提供波斯语搜索.
